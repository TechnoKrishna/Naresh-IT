6. Specificity demo (which rule wins?)

This text color is set by both .spec-demo (green) and #specId (red). The ID wins.

Specificity order: inline styles > IDs > classes/attributes/pseudo-classes > element selectors/pseudo-elements. When equal specificity, last declared rule wins.
